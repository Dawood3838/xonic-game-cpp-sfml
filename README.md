# Xonic Game

A Xonix-inspired arcade game developed in **C++ using SFML** as part of the Programming Fundamentals course at FAST NUCES Islamabad.

This project extends the classic Xonix gameplay by introducing multiple game modes, dynamic difficulty levels, enemy movement patterns, power-up mechanics, and persistent score tracking.

---

## Features

### Game Modes

* Single Player Mode
* Two Player Mode

### Difficulty Levels

* Easy
* Medium
* Hard
* Continuous Mode

### Gameplay Mechanics

* Territory capturing system
* Dynamic enemy movement
* Progressive difficulty scaling
* Score tracking
* Time tracking
* Move counter

### Power-Up System

* Earn power-ups by reaching score milestones
* Temporary gameplay advantages
* Strategic resource management

### Scoreboard

* Persistent score storage using file handling
* Top score tracking
* Score and time recording

### Menus

* Main Menu
* Level Selection Menu
* Results Screen
* Game Over Screen
* Scoreboard Screen

---

## Technologies Used

* C++
* SFML (Simple and Fast Multimedia Library)
* File Handling
* Object-Oriented Programming Concepts
* Event-Driven Programming
