# Xonic Game

A Xonix-inspired arcade game developed in C++ using SFML as part of the Programming Fundamentals course at FAST NUCES Islamabad.

## Features

* Single Player Mode
* Two Player Mode
* Difficulty Selection
* Dynamic Enemy System
* Power-Up Inventory
* Scoreboard System
* File-Based High Scores
* Enemy Movement Patterns
* Timer and Move Counter

## Technologies

* C++
* SFML
